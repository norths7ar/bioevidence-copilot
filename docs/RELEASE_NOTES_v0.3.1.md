# BioEvidence Copilot v0.3.1

v0.3.1 is a CI portability patch for the v0.3 fine-tuned extraction release.
Product behavior, annotations, model weights, splits, and evaluation results are
unchanged.

## Fixes

- Canonicalizes text provenance hashes to LF before SHA-256 calculation, making
  the tracked corpus manifest identical on Windows and Linux checkouts.
- Builds the QLoRA dry-run test dataset inside pytest's temporary directory
  instead of relying on ignored local `artifacts/` generated by an earlier run.
